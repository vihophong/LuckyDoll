# LUCKYDOLL
Unpacker, eventbuilder for AIDA

##  To Compile
cd LuckyDoll

cd ../

mkdir build_LuckyDoll

cd build_LuckyDoll

cmake ../LuckyDoll

make

## To run
./aida
or
./aidafull
or
./aida2tree
or
./aida2histcalib
or
./aida2histbkg

then follow the instruction
